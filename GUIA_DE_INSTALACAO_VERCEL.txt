MERCADOFLOW PDV - INSTALAÇÃO COM SUPABASE E VERCEL

1. Crie um projeto no Supabase.

2. No SQL Editor do Supabase, execute o arquivo:
   SUPABASE_SETUP.sql

3. No Supabase, abra Project Settings > API e copie:
   - Project URL
   - chave pública anon/publishable
   - chave secreta service_role

4. No Vercel, abra Settings > Environment Variables e crie:
   VITE_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY
   SUPABASE_SERVICE_ROLE_KEY

5. Em Authentication > URL Configuration no Supabase:
   - defina o domínio do Vercel como Site URL
   - adicione o domínio do Vercel em Redirect URLs
   - para desenvolvimento, adicione http://localhost:5173/**

6. Faça o deploy no Vercel.

7. Abra:
   https://SEU-DOMINIO.vercel.app/register

8. Crie o primeiro administrador.
   Depois do primeiro cadastro, novos usuários devem ser criados dentro da tela Usuários.

SEGURANÇA
- Nunca exponha SUPABASE_SERVICE_ROLE_KEY no frontend.
- A chave service_role não pode usar o prefixo VITE_.
- O banco já possui políticas RLS configuradas pelo arquivo SQL.
