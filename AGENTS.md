# Orientações do projeto

Este repositório contém o MercadoFlow PDV, uma aplicação React com Vite.

- Cliente de dados e autenticação: `src/api/client.js`.
- Configuração do frontend: `vite.config.js`.
- Configuração de publicação: `vercel.json`.
- Build: `npm run build`.
- Desenvolvimento: `npm run dev`.

Mantenha compatibilidade com implantação estática e com as rotas da SPA.
